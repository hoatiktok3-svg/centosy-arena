# STEP 01 - KIỂM TRA PROJECT VÀ THIẾT LẬP NỀN TẢNG

## Mục tiêu
Kiểm tra dự án hiện tại đang dùng công nghệ gì, app có chạy được không, và thiết lập nền tảng tối thiểu để tiếp tục làm các step sau.

## Việc cần làm
1. Kiểm tra cấu trúc thư mục hiện tại.
2. Xác định framework đang dùng: Next.js, React, Vite, Vue hay công nghệ khác.
3. Kiểm tra file package.json nếu có.
4. Chạy lệnh kiểm tra phù hợp:
   - npm install nếu thiếu node_modules
   - npm run dev hoặc lệnh tương ứng
   - npm run build nếu có thể
5. Ghi lại công nghệ, port chạy local và lỗi hiện tại nếu có.
6. Không sửa giao diện lớn trong step này.

## Kết quả mong muốn
- Biết project đang chạy bằng gì.
- Biết cách chạy app local.
- Biết lỗi hiện tại nếu có.
- PROJECT_STATUS.md được cập nhật rõ ràng.

## Không được làm
- Không thiết kế lại app.
- Không xóa file cũ.
- Không thêm tính năng mới.

## Sau khi hoàn thành
Cập nhật PROJECT_STATUS.md và PROMPT_QUEUE.md.
