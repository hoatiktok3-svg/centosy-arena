# STEP 16 - TẠO MÀN HÌNH QUẢN LÝ GAME CHO ADMIN

## Mục tiêu
Admin có màn hình xem và quản lý danh sách game hiện có.

## Yêu cầu chức năng
1. Hiển thị danh sách game:
   - Tên game
   - Mô tả
   - Trạng thái: active/inactive
   - Số lượt chơi
   - Điểm tối đa
2. Có nút bật/tắt trạng thái game bằng mock/local state.
3. Có nút xem câu hỏi/tình huống mẫu nếu đơn giản.
4. Có cảnh báo: dữ liệu hiện là mẫu nếu chưa có database.

## Việc cần làm
1. Tạo trang Admin Game Management.
2. Tạo mock data games.
3. Tạo component GameAdminCard.
4. Gắn route Admin bảo vệ.
5. Cập nhật dashboard link đến trang này.

## Kết quả mong muốn
- Admin nhìn được toàn bộ game.
- Có cảm giác hệ thống có thể quản trị được.

## Không được làm
- Không xây editor câu hỏi phức tạp trong step này.
- Không làm upload file/import Excel.
