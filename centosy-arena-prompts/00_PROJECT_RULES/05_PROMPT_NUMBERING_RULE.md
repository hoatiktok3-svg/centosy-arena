# RULE — Prompt Numbering Rule

Từ nay trong CENTOSY ARENA:
- Prompt tính năng chính dùng STEP và phải nối tiếp: STEP 85, STEP 86, STEP 87...
- Prompt sửa lỗi khẩn cấp dùng P0-xx.
- Prompt audit/hỗ trợ dùng AUX-xx.
- Prompt cập nhật/thư mục/quy trình dùng OPS-xx.
- Không tạo prompt không số thứ tự.
- Không đánh lại từ đầu.
- Nếu trùng số, không tự xóa file cũ; báo cáo và đề xuất merge.
