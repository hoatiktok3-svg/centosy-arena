# STEP 06 - TẠO DASHBOARD ADMIN CƠ BẢN

## Mục tiêu
Tạo trang quản trị cho Admin để xem tổng quan app nội bộ.

## Yêu cầu giao diện
Dashboard gồm các card KPI:
1. Tổng nhân viên.
2. Tổng lượt chơi hôm nay.
3. Tổng điểm đã ghi nhận.
4. Nhân viên top 1 hiện tại.
5. Game đang hoạt động.

## Yêu cầu chức năng
1. Chỉ Admin xem được.
2. Dữ liệu có thể dùng mock data.
3. Có link nhanh đến:
   - Quản lý nhân viên
   - Quản lý game
   - Bảng xếp hạng
   - Vinh danh
4. Giao diện đẹp, sạch, dễ xem trên desktop và mobile.

## Việc cần làm
1. Tạo trang Admin Dashboard.
2. Tạo các KPI card.
3. Gắn dữ liệu mẫu.
4. Đảm bảo route được bảo vệ bằng role Admin.

## Kết quả mong muốn
- Admin có trang tổng quan rõ ràng.
- App trông giống hệ thống quản trị thật hơn.

## Không được làm
- Không làm CRUD nhân viên chi tiết trong step này.
- Không làm module game trong step này.
