# STEP 03 - TẠO LAYOUT MOBILE-FIRST CƠ BẢN

## Mục tiêu
Tạo khung giao diện chính cho app nội bộ Centosy, ưu tiên nhân viên dùng trên điện thoại.

## Yêu cầu giao diện
1. Thiết kế mobile-first.
2. Có header đơn giản: tên app, trạng thái đăng nhập.
3. Có navigation phù hợp mobile:
   - Trang chủ
   - Game
   - Vinh danh
   - Xếp hạng
   - Tài khoản
4. Màu nhận diện ưu tiên:
   - Cam Centosy
   - Trắng
   - Xám đậm
   - Xám nhạt
5. Giao diện sạch, dễ bấm, chữ to vừa đủ.

## Việc cần làm
1. Tạo layout component dùng chung.
2. Tạo navigation mobile.
3. Tạo style cơ bản hoặc dùng Tailwind nếu project đã có.
4. Đảm bảo hiển thị ổn trên màn hình điện thoại.
5. Không thêm logic đăng nhập thật ở step này.

## Kết quả mong muốn
- App có khung giao diện mobile đẹp hơn.
- Các trang sau có thể dùng chung layout.

## Không được làm
- Không tạo game.
- Không tạo dashboard admin.
- Không thêm database.
