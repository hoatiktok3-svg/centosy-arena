# CENTOSY ARENA — PROJECT BRIEF

---

## Tên app
**CENTOSY ARENA**

---

## Mục tiêu
Web app nội bộ mobile-first cho Centosy Việt Nam, dùng để:
- Thi đua nội bộ
- Vinh danh thành viên xuất sắc
- Chơi game nội bộ
- Tính điểm & bảng xếp hạng
- Kết nối nhân sự toàn công ty

---

## Người dùng
Khoảng **80 thành viên**, chia theo các khối:

| Khối | Mô tả |
|---|---|
| Văn phòng | Bộ phận hành chính, kế toán, nhân sự... |
| Cửa hàng | Nhân viên bán lẻ tại các chi nhánh |
| Kho | Bộ phận kho vận, logistics |
| TMĐT | Thương mại điện tử, online sales |
| KDTT | Kinh doanh thị trường, B2B |

## MVP — Các màn hình cần có

| # | Tính năng | Mô tả ngắn |
|---|---|---|
| 1 | **Login** | Mock login, chọn nhân vật / khối |
| 2 | **Home Dashboard** | Điểm cá nhân, nhiệm vụ hôm nay, top nhanh |
| 3 | **Game Center** | Danh sách mini-game nội bộ |
| 4 | **Leaderboard** | Bảng xếp hạng theo tuần / tháng / khối |
| 5 | **Honor Wall** | Vinh danh thành viên xuất sắc |
| 6 | **Members** | Danh sách nhân sự, profile cơ bản |
| 7 | **Admin** | Quản lý điểm, duyệt nhiệm vụ, cấu hình |
| 8 | **Idea Bank** | Gửi ý tưởng cải tiến nội bộ |

## Quy tắc kỹ thuật

- ✅ Dùng **mock data** trước, chưa kết nối backend
- ✅ Không gọi AI API thật
- ✅ Mỗi bước chỉ làm **1 tính năng**
- ✅ Stack: Vite + React + TypeScript + Tailwind CSS
- ✅ Không đổi công nghệ khi chưa hỏi
